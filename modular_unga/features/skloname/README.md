# Склонялка слов

Как работает: вставляете русское слово обернув его в прок и на выходе получаете склонированное в другой падеж слово.

## Чуть подробное обьяснение...

Есть три рода - мужской (он), женский (она) и средний (оно)
Есть 7 падежей - Родительный (дефайн `RODITELNI`), Дательный (дефайн `DATELNI`), Винительный (дефайн `VINITELNI`), Творительный (дефайн `TVORITELNI`), Предложный (дефайн `PREDLOZHNI`)

```dm
skloname("слово на русском языке", дефайн падежа, род мужской или женский)
```

## Пример

```dm
// Начальное слово: ведро (средний род)
skloname("ведро", DATELNI) // Если у слова средний род то его не нужно указывать
// Итогое слово: ведру

// Начальное слово: computer (английское слово)
skloname("computer", DATELNI)
// Итогое слово: computer (потому что английские слова не обрабатываются)

// Начальное слово: пол (мужской род)
skloname("пол", TVORITELNI, "male") // Обратите внимание что если пол мужской или женский то это нужно указывать
// Итогое слово: полом
```
